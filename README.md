# SystemsEng-Team20-Website

Static technical report website for the Ambience-AI-1.5 university software engineering project.

## Pages

- Home
- Requirements
- Research
- Algorithms
- UI Design
- System Design
- Implementation
- Evaluation & Testing
- Conclusion
- Appendices

## Structure

- `index.html` and the section HTML files contain the page content.
- `assets/styles.css` contains the shared visual design.
- `assets/site.js` injects the shared navigation, footer, and page outline.

## Preview locally

Serve the folder with any static file server and open `index.html` through that server.

Example options:

- VS Code Live Server
- `python3 -m http.server`
- `ruby -run -e httpd . -p 8088`
